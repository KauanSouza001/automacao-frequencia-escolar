# 📌 Automação de Frequência Escolar com Python

## 🚀 Descrição

Este projeto automatiza o lançamento de frequência de alunos em um sistema escolar, eliminando tarefas manuais repetitivas.

A automação foi desenvolvida com Python utilizando a biblioteca PyAutoGUI, simulando interações humanas como cliques, navegação por teclado e preenchimento automático de campos.

---

## 🛠️ Tecnologias utilizadas

- Python
- PyAutoGUI

---

## ⚙️ Funcionalidades

- Preenchimento automático de frequência
- Navegação entre campos utilizando teclado
- Execução em múltiplas linhas automaticamente
- Redução de tarefas repetitivas

---

## 💡 Contexto

Este projeto foi desenvolvido com base em uma necessidade real em ambiente de secretaria escolar, onde o lançamento de dados era feito manualmente.

A automação permitiu aumentar a eficiência e reduzir o tempo gasto nessas atividades.

## 🧭 Captura de Coordenadas

Este projeto utiliza posições específicas da tela para funcionar.

Para facilitar isso, foi criado um script auxiliar para capturar coordenadas.
### ▶️ Como usar

1. Execute o arquivo: python capturar_coordenadas.py
2. Mova o mouse até o local desejado  
3. Aguarde alguns segundos  
4. As coordenadas serão exibidas no terminal  

Depois, basta substituir os valores no código principal.

---

## ⚠️ Observações

- O projeto utiliza coordenadas fixas da tela  
- Pode ser necessário ajustar conforme resolução ou sistema utilizado  

---

## ▶️ Como executar o projeto

1. Instale o Python  
2. Instale a biblioteca necessária: pip install pyautogui
3. Execute o script principal: python automacao_frequencia_escolar.py
